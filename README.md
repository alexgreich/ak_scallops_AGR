# ak_scallops
File structure:

1) code - contains subdirectories for (mostly) R code by topic
2) data - contains subdirectories with various data files, available on the Kodiak intranet
3) output - contains subdirectories for analytical outputs
4) figures - contains subdirectories for figures relating to various analyses
5) safe - contains each NPFMC assessment cycle SAFE materials, ex Rmd scripts for SAFEs and appendices
